xtracer
=======


About
-----
	Xtracer is a basic raytracer written in c and c++.

Features
--------

	Primitives:
		- Plane
		- Triangle
		- Sphere

	Materials:
		- Lambert
		- Phong

	Lights:
		- Point light

	Cameras:
		- Pinhole camera
	
	Drivers:
		- SDL
		- PPM

Dependencies
------------
	libsdl    
	libnmath   https://github.com/4rknova/libnmath
	libnparse  https://github.com/4rknova/libnparse
